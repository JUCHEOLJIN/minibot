# 내장 스킬 디렉토리

이 디렉토리는 **봇에 포함되는 내장 스킬** 공간입니다.

현재는 비어 있습니다. mini-bot은 built-in 스킬 없이 시작합니다.

---

## 사용자 스킬 추가

내장 스킬 대신 **사용자 스킬**을 `~/.mini-bot/skills/` 에 추가하세요.

자세한 작성 방법은 봇이 자동 생성하는 `CLAUDE.md`를 참고하세요.

---

## 내장 스킬을 만들고 싶다면

이 디렉토리에 폴더를 만들면 봇 시작 시 자동으로 로드됩니다.

```
skills/
└── my-builtin-skill/
    ├── SKILL.md
    └── my-builtin-skill.js
```

사용자 스킬과 이름이 겹치면 **사용자 스킬이 우선**합니다.
